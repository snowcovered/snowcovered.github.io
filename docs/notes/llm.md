## 模型建立

Q1: 度量为1的任务集合 $i \in [0,1]$是什么意思？任务不应该是一个一个的吗？在$[0,1]$这样的区间上一个点不就测度为零了嘛？

- **含义**：这里的“任务集合”是一个**连续体**（continuum），而不是一堆离散任务。
- 区间 $$[0,1]$$ 代表所有任务的“索引”，整体的勒贝格测度为 1，表示**总任务量归一化为 1**。
- 单个点的测度确实是 0，但我们关心的是区间上的**积分**（如 $$\int_0^1 w_i v(x_i,y_i,z)\,di$$），而不是单点求和。
- 这种连续任务的设定在机制设计或最优控制中很常见，用来表示“无限多且可分”的需求。

Q2：交叉偏导数大于0有什么意义？

- 这些是**边际互补效应**：
  * $v_{xy}>0$：输入 token 越多时，增加输出 token 的边际收益越大（反之亦然）。
  * $v_{xz}>0$：输入 token 越多，微调带来的边际收益越大。
  * $v_{yz}>0$：输出 token 越多，微调带来的边际收益越大。
- 经济学解释：**投入之间是互补的**。  
  例如，提供更多上下文（输入）时，多生成输出或多做微调更有价值。

Q3:$\alpha+\beta+\gamma<1$这个要求是为什么？

- 在 Cobb–Douglas 生产函数  
  $v(x,y,z)=x^{\alpha} y^{\beta} (b+z)^{\gamma}$  
  中，$\alpha+\beta+\gamma$ 是**总产出相对于所有投入的弹性之和**。
- $\alpha+\beta+\gamma<1$  **规模报酬递减**（decreasing returns to scale）：
  - 若所有投入都放大 $\lambda>1$，产出增加 $\lambda^{\alpha+\beta+\gamma}<\lambda$倍。
  - 保证：
    * 产出不会无限膨胀；
    * 买方不会因无限投入而获得无穷收益；
    * 模型的最优解稳定可行。
- 这与“单个任务的投入存在边际报酬递减”的假设一致
